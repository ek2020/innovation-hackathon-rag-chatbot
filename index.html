RAG Workflow Project

Retrieval Augmented Generation with Azure OpenAI and Qdrant

Project Overview

This project demonstrates a comprehensive implementation of a Retrieval Augmented Generation (RAG) workflow. RAG enhances large language models by retrieving relevant information from a knowledge base before generating responses, combining the reasoning capabilities of LLMs with the accuracy and specificity of your own data.
